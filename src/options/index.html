Available Goodreads

Easy setup:
To get started, find your local library on Overdrive's library search map. Simply click "Add this library to Available Goodreads" when you find your local library.

Setup complete! Go to your to-read bookshelf to see which books are available!

Detailed information:
This extension will add an "Availability on Overdrive" listing to each GoodReads book description page and an "on overdrive" column to each bookshelf. A legend of terms and symbols used:
3 available3 copies are available
4/2 holds4 people waiting for 2 copies of the book
0/1 holdsNo one waiting on the 1 copy checked out
not foundNo results found searching by book title and author
Audiobook
Hover over each listing to see the title and author of the result. Click the listing to open the Overdrive search result page in a new window.

Manually adding a library:
If by searching on Overdrive's library search you were unable to find your local library's URL, you could try to manually find it with the following steps.

Note: Your library URL must contain .lib.overdrive.com. Many major libraries hide this URL, but you can usually figure it out by simply looking at the domain name. For example:
CityLibrary ebook siteActual Overdrive URL
NYChttp://ebooks.nypl.org/http://nypl.lib.overdrive.com/
LAhttp://e-media.lapl.org/http://lapl.lib.overdrive.com/
Chicagohttp://www.chipublib.org/http://chipublib.lib.overdrive.com/

If you're still having difficulty, then you can also check the source of the library site for the URL.
  1. Go to your local library's Overdrive page
  2. Press Ctrl+U to view the source.
  3. Press Ctrl+F and search for: lib.overdrive
  4. Copy the URL found into the text box above.
